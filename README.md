# Генератор кадровых приказов

## Возможности
- Прием / увольнение / отпуск
- Русский + казахский
- Генерация DOCX
- Работает без сервера (GitHub Pages)

## Запуск
Просто открой index.html или задеплой на GitHub Pages

## Деплой
1. Создать репозиторий
2. Залить файлы
3. Settings → Pages → main branch
