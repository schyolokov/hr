# Генератор кадровых приказов

Веб-приложение для формирования кадровых приказов ТОО «СТ Эсэмбли» на русском и казахском языках в одном DOCX-файле.

## Типы приказов

- Ежегодный оплачиваемый отпуск (неиспользованная часть)
- Ежегодный оплачиваемый отпуск (за фактически отработанный период)
- Отзыв из ежегодного отпуска с компенсацией
- Отпуск с сохранением заработной платы (по коллективному договору)

## Возможности

- Формат приказов соответствует образцам из кадровой документации
- Двуязычная генерация: русский + казахский (на отдельных страницах)
- Склонение ФИО (русский — petrovich, казахский — упрощённая морфология)
- Автонумерация приказов через localStorage
- Работает без сервера (GitHub Pages)

## Запуск

Откройте `index.html` в браузере или опубликуйте на GitHub Pages.

## Деплой

1. Загрузите файлы в репозиторий `schyolokov/hr`
2. Settings → Pages → Deploy from branch `main`
3. Сайт будет доступен по адресу `https://schyolokov.github.io/hr/`
